Form

사용자로부터 어떠한 값을 받을 목적으로 기술 전달될 데이터는 form 태그에 포함되어야 한다.
GET: 256Byte - 4000Byte 이내의 크기의 데이터만 전송 가능.
주소줄에 address?key=value&key=value...
POST: 크기 상관없음.
body 처음에 붙어서 넘어간다. (내용을 볼 수 없음)

상품 주문
size는 필드의 크기(입력할 수 있는 칸의 크기)
maxlength 정의한 필드에 입력할 수 있는 최대 값. 10이면 10자만 입력 가능. size가 20이면 크기는 20자 입력이 가능한데 입력 할 수 있는 최대 값은 10자만 입력 가능
disabled는 값을 입력해주더라도 JSP에게 값을 전달하지 않는다.
hidden은 웹 화면 상 보이지는 않지만 입력된 값이 있다면 JSP에게 값을 전달한다. (사용자는 모르게 서버에 데이터를 전달하려고 할때사용) 주문한 사람




배송지







radio, checkbox를 하나로 묶은형태 (단일선택과 다중선택을 묶음)
option은 예외적으로 select value속성이 없더라도 작성한 문자열이 전송된다.
